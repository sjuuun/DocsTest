# DocsTest
Sphinx layout test for documentation

## How to use

### Install Sphinx
Need Python 3.5+

```bash
$ pip install sphinx
```

### Git clone
```bash
$ git clone https://github.com/sjuuun/DocsTest.git
```

### Make html
```bash
$ DocsTest/docs

# install theme
$ pip install sphinx-rtd-theme

# make html file and open
$ make html
$ open build/html/index.rst
```
